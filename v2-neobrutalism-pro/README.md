# v2 - Neo-Brutalism Pro

Улучшенная версия с более проработанным дизайном.

## Запуск

```bash
python app.py
```

Сайт будет доступен на `http://localhost:25616`

## Улучшения относительно v1

- Оптимизированный код сервера
- Улучшенная типографика
- Расширенное описание услуг
- Доработанная цветовая схема

## Изменения

- ThreadingHTTPServer для лучшей производительности
- Улучшенная обработка ошибок
- Чище структурированный HTML
