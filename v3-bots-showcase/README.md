# v3 - Bots Showcase Landing

Полноценный landing page для презентации Telegram ботов.

## Запуск

```bash
python app.py
```

## Особенности

- Фокус на презентации ботов
- Детальные описания каждого продукта
- Современные анимации
- CTA кнопки для каждого бота
- Профессиональный дизайн

## Боты

1. **Network Monitor Bot** - мониторинг доступности сайтов
2. **Stresser Bot** - стресс-тестирование
3. **AI Assistant** - умный помощник
